# Company

Minimal static site starter. Placeholder page until the company has a name.

## Local preview

```bash
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

1. Push this repo to GitHub (or connect the folder in Vercel).
2. Import the project in [Vercel](https://vercel.com/new).
3. No build command needed — static files deploy as-is.
4. Deploy from `main`.

## Structure

| File | Purpose |
|------|---------|
| `index.html` | Placeholder landing page |
| `styles/tokens.css` | Design tokens (colors, motion) |
| `favicon.svg` | Placeholder favicon |
| `vercel.json` | Vercel static deploy config |
